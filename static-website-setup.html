üîß Detailed Setup

Prerequisites

  • GitHub account dengan public repository
  • Git installed dan configured
  • Python 3.x installed

Step 1: Setup GitHub Repository

cd /workspaces/ebooks
git init
git add .
git commit -m "Initial commit with static website"
git remote add origin https://github.com/YOUR_USERNAME/ebooks.git
git branch -M main
git push -u origin main

Step 2: Enable GitHub Pages

  1. Go to GitHub repository
  2. Settings ‚Üí Pages
  3. Source: Select "Deploy from a branch"
  4. Branch: Select "main" and "/ (root)"
  5. Click "Save"

Step 3: Deploy Static Website

Manual Method:

# Generate manifest
python3 generate_books_manifest.py

# Test locally
python3 -m http.server 8000

# Rename file untuk GitHub Pages
cp index-github-pages.html index.html

# Commit dan push
git add .
git commit -m "Add static website"
git push origin main

# Wait 1-2 minutes untuk GitHub Pages build
# Website available di: https://YOUR_USERNAME.github.io/ebooks/

Automated Method:

# Run deployment script (recommended)
chmod +x deploy-to-github-pages.sh
./deploy-to-github-pages.sh YOUR_USERNAME
‚ÑπÔ∏è Script otomatis akan create gh-pages branch, copy files, dan push ke GitHub

üêõ Troubleshooting

Website not loading?

  • ‚úÖ Clear browser cache (Ctrl+Shift+Delete)
  • ‚úÖ Check branch is set to main/gh-pages
  • ‚úÖ Verify rename index-github-pages.html ‚Üí index.html

PDFs not downloading?

  • ‚úÖ Check EBOOKS/ dan pengetahuan/ folders pushed
  • ‚úÖ Verify file paths in books_data.js correct
  • ‚úÖ Check GitHub Pages settings

  • ‚úÖ First load may take 30s, subsequent loads faster

‚ö° Performance Tips

Reduce Repository Size

Jika repository terlalu besar, gunakan Git LFS:

# Install Git LFS
sudo apt-get install git-lfs

# Track PDF files
git lfs track "*.pdf"

# Push changes
git add .gitattributes
git commit -m "Setup Git LFS"
git push

Optimize PDFs

Reduce PDF size dengan compression:

# Linux/Mac
gs -sDEVICE=pdfwrite -dCompatibilityLevel=1.4 \
   -dPDFSETTINGS=/ebook -dNOPAUSE -dQUIET -dBATCH \
   -sOutputFile=output.pdf input.pdf

Use CDN

Untuk kecepatan lebih, deploy ke Terraform atau Cloudflare:

  • Netlify - Free, fast deploys
  • Vercel - Optimized for performance
  • Cloudflare Pages - Global CDN
